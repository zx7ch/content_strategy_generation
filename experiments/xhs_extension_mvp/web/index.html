XHS 搜索与采集仪表盘

围绕同一个搜索主题，把操作、拓展搜索、热点追踪、采集概览和推荐笔记拆成独立模块。任何一个模块刷新失败，都不会影响其他区域继续使用。

用户操作区域

输入主题并创建任务。插件会自动识别当前任务，采集后工作台会自动刷新快照。

热点追踪区域

基于用户原始搜索词，直接调用 spider 的真实排序接口拉取“最多点赞 / 最多评论 / 最多收藏”结果。创建任务后会自动刷新一次，失败时保留上次成功结果。
